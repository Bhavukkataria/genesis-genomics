5-Kingdom Classification

Robert Whittaker proposed dividing all life into Monera, Protista, Fungi, Plantae, and Animalia based on cell type, nutrition mode, and body organization. Monera are prokaryotes; all others are eukaryotes.

Cell Division

Mitosis: The Replication Dance

Prophase → Metaphase → Anaphase → Telophase

Produces 2 genetically identical daughter cells. Essential for growth, repair, and asexual reproduction. Preceded by S phase (DNA replication) during interphase.

Meiosis

Meiosis: Genetic Shuffle

Two rounds of division (Meiosis I & II) yield 4 haploid gametes. Crossing over during Prophase I generates genetic diversity — nature's stochastic recombination operator.

Stem Cell Applications

Regenerative Medicine

Stem cells are undifferentiated cells with unlimited self-renewal capacity and the ability to differentiate into specialized cell types. They are classified by potency:

Totipotent — can become any cell type (zygote)
Pluripotent — most body cells (embryonic stem cells)
Multipotent — specific tissue types (bone marrow)
Applications: Parkinson's disease, spinal cord injury, diabetes, cardiac repair, leukaemia treatment via bone marrow transplants.

genetic_algorithm.c C-Logic of Life 
/*
 * THE C-LOGIC OF LIFE
 * Genetic Algorithm — mimicking evolutionary selection in code.
 * Biological parallel: Population → Selection → Crossover → Mutation → Next Gen
 */

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <time.h>

#define POP_SIZE    50   // Population of organisms
#define GENE_LEN    16   // Length of genetic sequence (chromosome)
#define TARGET      0xABCD // The 'optimal phenotype' we're evolving toward
#define MUT_RATE    0.02f // 2% mutation rate per gene bit

/* === CHROMOSOME STRUCTURE ===
 * Analogous to a single organism's haploid genome
 */
typedef struct {
    unsigned int chromosome;   // Binary gene sequence
    float        fitness;       // Survival score (higher = more fit)
} Organism;

/* === STEP 1: INITIALIZATION ===
 * Like Abiogenesis — random primordial chemistry creates first genomes
 */
void init_population(Organism population[]) {
    srand((unsigned)time(NULL));
    for (int i = 0; i < POP_SIZE; i++) {
        population[i].chromosome = rand() % (1 << GENE_LEN);
        population[i].fitness    = 0.0f;
    }
}

/* === STEP 2: FITNESS FUNCTION ===
 * Nature's selection pressure: count matching bits to TARGET
 * More matching bits → higher fitness → more likely to reproduce
 */
float evaluate_fitness(unsigned int chrom) {
    unsigned int diff     = chrom ^ TARGET; // XOR → differing bits
    int          matching = GENE_LEN - __builtin_popcount(diff);
    return (float)matching / GENE_LEN;    // Normalize [0.0, 1.0]
}

/* === STEP 3: SINGLE-POINT CROSSOVER ===
 * Sexual reproduction: splice two parent genomes at a random locus.
 * Produces genetic novelty — the source of heritable variation.
 */
unsigned int crossover(unsigned int parent_A, unsigned int parent_B) {
    int          locus  = rand() % GENE_LEN; // Random crossover point
    unsigned int mask   = (1 << locus) - 1;
    // Inherit left-segment from A, right-segment from B
    return (parent_A & ~mask) | (parent_B & mask);
}

/* === STEP 4: MUTATION ===
 * Cosmic radiation, replication errors → random bit flips
 * Low probability, but the engine of long-term evolution
 */
unsigned int mutate(unsigned int chrom) {
    for (int bit = 0; bit < GENE_LEN; bit++) {
        if ((float)rand() / RAND_MAX < MUT_RATE)
            chrom ^= (1 << bit); // Flip the bit — point mutation!
    }
    return chrom;
}

/* === MAIN: GENERATIONAL LOOP ===
 * Simulate N generations of evolutionary pressure
 */
int main() {
    Organism pop[POP_SIZE], next_gen[POP_SIZE];
    init_population(pop);

    for (int gen = 0; gen < 500; gen++) {

        /* Evaluate all organisms */
        float best = 0.0f;
        for (int i = 0; i < POP_SIZE; i++) {
            pop[i].fitness = evaluate_fitness(pop[i].chromosome);
            if (pop[i].fitness > best) best = pop[i].fitness;
        }

        printf("Gen %03d | Best fitness: %.2f%%\n", gen, best * 100);
        if (best >= 1.0f) { printf("Optimal genotype reached!\n"); break; }

        /* Tournament selection + crossover → next generation */
        for (int i = 0; i < POP_SIZE; i++) {
            int a = rand() % POP_SIZE, b = rand() % POP_SIZE;
            Organism *pA = (pop[a].fitness > pop[b].fitness) ? &pop[a] : &pop[b];
            int c = rand() % POP_SIZE, d = rand() % POP_SIZE;
            Organism *pB = (pop[c].fitness > pop[d].fitness) ? &pop[c] : &pop[d];

            next_gen[i].chromosome = mutate(crossover(pA->chromosome, pB->chromosome));
        }
        memcpy(pop, next_gen, sizeof(pop));
    }
    return 0;
}
CRISPR-Cas9 Spotlight

Editing the Genome's Source Code

CRISPR-Cas9 acts as molecular scissors guided by RNA to precise genomic locations. It enables deletion, replacement, or correction of disease-causing mutations. The 2020 Nobel Prize in Chemistry recognised Jennifer Doudna & Emmanuelle Charpentier for this breakthrough — effectively making biology programmable.
